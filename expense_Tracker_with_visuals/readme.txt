# 💸 Expense Tracker with Visuals

## 📌 Description

A simple expense tracking application built using **Python, Pandas, Matplotlib, and Streamlit**.
It allows users to record expenses, visualize spending patterns, and monitor budgets.

---

## ⚙️ Technologies Used

* Python
* Pandas
* Matplotlib
* Streamlit

---

## 📂 Files

* `app.py` – Main application
* `data/expenses.csv` – Expense data
* `requirements.txt` – Dependencies

---

## 🚀 How to Run

```bash
pip install -r requirements.txt
streamlit run app.py
```

---

## 🧠 Features

* Add and store expenses
* Categorize spending
* Pie chart (category-wise)
* Bar chart (date-wise)
* Budget alert system
* Export data to Excel

---

## 📌 Usage

1. Enter expense details in sidebar
2. Click **Add**
3. View charts and insights
4. Set budget to track spending
