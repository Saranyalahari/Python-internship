# 🛒 Product Recommendation System

## 📌 Description

A simple product recommendation system built using **Python, Pandas, Scikit-learn, and Streamlit**.
It suggests products based on similarity and user ratings.

---

## ⚙️ Technologies

* Python
* Pandas
* Scikit-learn
* Streamlit

---

## 📂 Files

* `app.py` – Streamlit UI
* `recommender.py` – Recommendation logic
* `data/` – Dataset files
* `requirements.txt` – Dependencies

---

## 🚀 How to Run

```bash
pip install -r requirements.txt
streamlit run app.py
```

---

## 🧠 Features

* Content-based filtering
* Collaborative filtering
* Hybrid recommendations
* Export results to CSV

---

## 📌 Usage

1. Select a product
2. Click **Recommend**
3. View suggested products
